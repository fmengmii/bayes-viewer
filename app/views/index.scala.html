The Bayesian Network Viewer is a tool for viewing and sharing clinical Bayesian networks. @** as well as various other types of predictive models. **@


For Public Bayesian Network, any user can view it.

For Private Bayesian Network, only registered user can view it. The registered user can upload new network files and shared them with other registered users. Also the owner of the network file can update and delete the network file, and view the log history of the network file.


The file format for the bayesian networks is *.xdsl that is built based on GeNIe.


Any question or comment please contact us.
