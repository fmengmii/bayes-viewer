The Bayesian Network Viewer is a tool for viewing and sharing clinical Bayesian networks. @** as well as various other types of predictive models. **@


For Public Bayesian Network, any user can view the model and raw data, inference, and test own raw data using the model.

For Private Bayesian Network, only registered user can view the model and raw data, inference, and test own raw data using the model and learn a new model using the testing data or combined data with original one. The registered user can upload new models and shared them with other registered users. Also the owner of the model can update and delete the model, and view the log history of the model.


The file format of the bayesian networks is *.xdsl that is built on GeNIe.


Any question or comment please contact us.
